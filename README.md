## Demonstration from 3 private repositories ##

### 1. Computer Vision: Multiclass Semantic Segmentation of cartridge images ###
   
The purpose of this repo is automated detection of Region of Interest (Breech Face/BF and Firing Pin/FP) in cartridge case images using deep learning - U-Net model and FCNs model, automated draw direction arrow from FP to firing pin drag by cv2.drawContours.

IoU         |    U-Net   |   FCNs
------------|------------|-----------
 Background |   99.27%   |  98.74%
 BF     |   95.98%   |  93.75%
 FP     |   87.29%   |  82.39%


 
<img src="https://github.com/fttlzyx/Computer-Vision_Multiclass-Semantic-Segmentation/assets/79032474/91cc5d10-b834-4b5a-b2c6-5605277eccfc" width = "70%">

_Figure 1 - Random selected sample prediction result from U-Net Model_

<img src="https://github.com/fttlzyx/Computer-Vision_Multiclass-Semantic-Segmentation/assets/79032474/873e535d-05b1-4e21-909f-25d9088eed0a" width="70%">

_Figure 2 - Random selected sample prediction result from FCNs Model_

FP and firing pin drag will be automatically detected by cv2.findContours. The center of each contour will be located and an arrow line will be draw from the center for FP to the center of firing pin drag by cv2.arrowedLine.
